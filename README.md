## Steps

- ## Install Packages
  ```sh
    npm install
    npm install --legacy-peer-deps     # if dependency conflitcs
  ```
- ## Run App

  ```sh
    npm start
  ```

- ## Build App
  ```sh
    npm run build
  ```
